## 1. 문서 개요

### 1.1 목적

본 정책서는 AX Flow 시스템의 **화면별 기능 동작 규칙, 데이터 처리 기준, 상태 전이 조건, 예외 처리 방안**을 정의한다. PRD(제품 요구사항 정의서)에서 정의된 화면 구성요소가 **어떤 조건에서, 어떻게 동작하며, 어떤 제약사항을 가지는지**를 명세하여 개발·QA·운영 간 합의 기준으로 활용한다.

### 1.2 용어 규칙

- **MUST** — 반드시 구현해야 하는 필수 요건
- **SHOULD** — 권장 사항으로, 합당한 사유가 있을 경우 생략 가능
- **MAY** — 선택적 구현 사항

| CHAT-P-09 | 근거 문서 인용 | 모든 AI 응답 하단에 참조 문서명과 절/청크 번호를 인용 블록으로 표시한다. 인용 블록 클릭 시 해당 문서 상세 화면의 원본보기 탭으로 이동한다. |

| CHAT-P-11 | 피드백 수집 | 각 AI 응답마다 좋아요/아쉬워요 버튼과 후속 질문을 통해 사용자가 피드백을 유도한다. 사용자가 피드백을 하면 해당 응답의 질의-응답-피드백 데이터를 기록하여 향후 개선에 활용한다. |
