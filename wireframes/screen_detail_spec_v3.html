본 화면설계서에서 사용되는 주요 용어 및 약어의 정의

용어영문정의
GraphRAGGraph-based Retrieval Augmented Generation지식 그래프와 벡터 검색을 결합하여 LLM 답변의 정확도와 맥락 이해도를 향상시키는 검색 증강 생성 기법
청크 (Chunk)Chunk문서를 일정 단위로 분할한 텍스트 조각. 벡터 임베딩과 검색의 기본 단위가 됨
온톨로지Ontology특정 도메인의 개념(엔티티), 속성, 관계를 체계적으로 정의한 스키마. 지식 그래프 구축의 기반
엔티티Entity지식 그래프에서 고유한 대상(장비, 공정, 재료, 인물 등)을 나타내는 노드
MCPModel Context ProtocolLLM과 외부 시스템 간 표준화된 통신 프로토콜. AI 에이전트가 외부 도구(API, DB)를 호출할 수 있게 함
RBACRole-Based Access Control사용자의 역할(등급)에 따라 시스템 자원에 대한 접근 권한을 제어하는 보안 모델
RagasRagas FrameworkRAG 시스템의 성능을 정량적으로 평가하는 오픈소스 프레임워크. Faithfulness, Answer Relevancy, Context Precision, Context Recall 등의 지표를 제공
FaithfulnessFaithfulnessAI 답변이 제공된 컨텍스트(검색된 문서)에 충실한 정도. 환각(hallucination) 없이 근거에 기반한 답변을 생성하는지 측정
Answer RelevancyAnswer RelevancyAI 답변이 사용자의 질문에 얼마나 관련성이 높은지 측정하는 지표
Context PrecisionContext Precision검색된 컨텍스트 중 답변 생성에 실제로 유용한 정보의 비율
Context RecallContext Recall정답에 필요한 정보가 검색된 컨텍스트에 얼마나 포함되어 있는지 측정
TemperatureTemperatureLLM의 응답 무작위성 조절 파라미터. 0에 가까울수록 결정적, 높을수록 창의적/다양한 응답 생성
Top-KTop-K검색 결과 상위 K개의 청크를 LLM 컨텍스트에 포함시키는 파라미터
Top-PTop-P (Nucleus Sampling)누적 확률이 P 이하인 토큰들만 샘플링 후보로 사용하는 파라미터
Score ThresholdScore Threshold검색 결과의 최소 유사도 점수. 이 점수 미만의 결과는 컨텍스트에서 제외됨
하이브리드 검색Hybrid Search벡터 유사도(Vector) 검색과 그래프 관계성(Graph) 검색을 결합하여 최적의 검색 결과를 도출하는 방식
커뮤니티 감지Community Detection그래프에서 서로 밀접하게 연결된 노드 그룹(클러스터)을 자동으로 식별하는 알고리즘
실체 연결Entity Resolution동의어, 약어, 코드명 등 동일 대상을 지칭하는 서로 다른 용어를 하나의 노드로 통합하는 과정
OCROptical Character Recognition이미지나 스캔 문서에서 텍스트를 자동으로 인식·추출하는 기술
파이프라인Pipeline문서 처리의 전체 흐름: 파싱(OCR) → 청킹 → 엔티티/관계 추출 → 그래프 적재
sLLMSmall Large Language Model경량화된 대규모 언어 모델. 토큰 비용 절감을 위해 특정 작업에 특화된 소형 모델 사용
SkillSkillAI 에이전트가 수행할 수 있는 특정 능력. .md 파일로 정의되며 관련 Tool/MCP와 연결됨
HookHookAI 에이전트의 질문 처리 전(Before) 또는 후(After)에 자동으로 실행되는 처리 로직
CommandCommand채팅에서 / 접두어로 실행하는 특수 명령어. 특정 Skill이나 기능을 즉시 호출
